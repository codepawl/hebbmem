# hebbmem Visual Assets

Auto-generated images for README, social media, and documentation.

## Regenerate

```bash
uv pip install hebbmem[assets]
uv run python scripts/generate_assets.py
```

## Usage

### README
- `readme/hero_banner.png` — top of README
- `readme/how_it_works.png` — 3 mechanisms explained
- `readme/benchmark_results.png` — benchmark bar chart
- `readme/recall_pipeline.png` — recall pipeline flowchart
- `readme/graph_example.png` — example memory graph

### Social Media

- `social/x_thread_*.png` — X/Twitter thread images (1600x900)
- `social/devto_cover.png` — Dev.to article cover (1000x420)
